# InterviewIQ

A resume-aware interview intelligence system that uses retrieval-augmented generation, answer evaluation, and job-description matching to improve interview preparation.

## Planned Features
- Resume upload and parsing
- Resume-aware interview answer generation
- Answer feedback scoring
- Job description matching
- Analytics dashboard
- Retrieval experiments

## Tech Stack
- FastAPI
- React
- OpenAI API
- FAISS
- Python
- Tailwind CSS
